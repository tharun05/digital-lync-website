What is this course about ?

Django is a Web Development Python Web Framework. It helps in building the Web Application with best quality FrameWork. Django supports MVC (Model-View-Controller) Pattern and MVT (Model-View-Template) Pattern. Django is written in pure Python Code.Django is stable and is constantly moving forth with new future and improvements.
