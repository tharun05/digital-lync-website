What is this course about ?

The purpose of the course is for students to learn how to design , prototype , and evaluate the user interface using different types of methods.

What you will learn

  • Fundamentals of website Building.
  • Building JavaScript Based Application.
  • Concepts and Patterns of JavaScript Programming language.
  • Understanding how JavaScript library is built.
  • Learning HTML5 and CSS3
  • Building responsive websites using Bootstrap.
  • Understanding Differences between JS Library and JS Frameworks
  • Building a web application using Angular JS
  • How to work on a team project and how to hand on the results of work both in written & oral form.
