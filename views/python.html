What is this course about ?

This course is focused to design to impart an in-depth knowledge of the various data analytic techniques which are performed using R. The course is designed with real time projects, case studies and R cloud labs to practise.

What you will learn

  • The students will get the basic understandings of business analytics
  • Gain foundational understandings of various statistical concepts
  • Let you know the use of linear, non-linear regression models, and classification techniques for data analysis
  • To know how to use clustering methods including K-means, DBSCAN and hierarchical clustering.
  • Learn how to use various association rules and Apriori algorithm.
