What is this course about ?

This course is all enthusiasts who are willing to build full stack javascript applications with mean stack using Mongodb, Node JS, Angularjs, Express JS and React JS.

What you will learn

  • Building full stack JavaScript Applications
  • Learn how Node.js is architected to allow high scalability with asynchronous code
  • Create basic web applications with Node.js
  • Build an HTTP server using the core modules in Node.js
  • Test the reliability of the application with unit tests
  • Make Angular work with your chosen UI widget framework
  • Write more sophisticated end-to-end tests

Program Syllabus
MONGO DB

Mongodb is an Open Source and Nosql database. It is cross-platform and its document oriented database is written in C++. The Mongodb provides best performance, good availability and auto scaling. The Mongodb is used in Big Data, Content Management System, Mobile and Social Infrastructure, User Data Management and Data Hub.

view cirriculum
  • Module 1 – Design Goals, Architecture and Installation
  • Module 2 – CRUD Operations
    • 1. MongoDB Development Architecture,
    • 2. MongoDB Production Architecture,
    • 3. MongoDB CRUD Introduction,
    • 4. MongoDB CRUD Concepts,
    • 5. MongoDB CRUD Concerns (Read & Write Operations)
    • 6. Concern Levels
    • 7. Journaling etc., Cursor
    • 8. Query Optimizations
  • Module 3 – Schema Design and Data Modelling
    • 1. Data Modelling Concepts,
    • 2. Type of Data Modelling,
    • 3. Why Data Modelling? Data Modelling Approach,
    • 4. Analogy between RDBMS & MongoDB Data Model,
    • 5. MongoDB Data Model (Embedding & Linking),
    • 6. Challenges for Data Modelling in MongoDB,
    • 7. Data Model Examples and Patterns,
    • 8. Model Relationships between Documents: Model One-to-One Relationships with Embedded Documents,
    • 9. Model One-to-Many Relationships with Embedded Documents,
    • 10. Model One-to-Many Relationships with Document References,
    • 11. Model Tree Structures: Model Tree Structures with Parent References,
    • 12. Model Tree Structures with Child References,
    • 13. Model Tree Structures with an Array of Ancestors,
    • 14. Model Tree Structures with Materialized Paths,
    • 15. Model Tree Structures with Nested Sets,
    • 16. Model Specific Application Contexts: Model Data for Atomic Operations
    • 17. Model Data to Support Keyword Search,
    • 18. Data Model References, Use Case of Data modelling.
  • Module 4 – Marking Up Text
    • 1. Administration Concept in MongoDB,
    • 2. Health Check of MongoDB Database,
    • 3. Monitoring of Various issues related with Database,
    • 4. Monitoring at Server, Database,
    • 5. Collection Level, and Various Monitoring Tools related to MongoDB,
    • 6. Database Profiling, Locks, Memory Usage, No of connections, page fault etc.,
    • 7. Backup and Recovery Methods for MongoDB,
    • 8. Export and Import of Data to and From MongoDB,
    • 9. Run time configuration of MongoDB, Production Notes/Bets Practices,
    • 10. Data Managements in MongoDB (Capped Collections/ Expired data from TTL),
    • 11. Hands on Administrative Tasks.
  • Module 5 – Scalability and Availability
    • 1. Introduction to Replication (High Availability),
    • 2. Concepts around Replication,
    • 3. What is ReplicaSet and Master Slave Replication? Type of Replication in MongoDB,
    • 4. How to setup a replicated cluster, Managing ReplicaSets etc.,
    • 5. Introduction to Sharding (Horizontal Scaling), Concepts around Sharding,
    • 6. What is shards, Key, ConfigServer,
    • 7. Query Router etc.? How to setup a Sharding
    • 8. Type of Sharding (Hash Based, Range Based etc.), and Managing Shards.
  • Module 6 – Indexing and Aggregation Framework
  • + Module 7 – Application Engineering and MongoDB Tools
  • Module 8 – Project, Additional Concepts and Case Studies
